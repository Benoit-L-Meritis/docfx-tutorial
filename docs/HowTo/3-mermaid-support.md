# Support natif de Mermaid

DocFX prend en charge les diagrammes au format Mermaid.

> [!NOTE]
> Mermaid diagrams is only supported in the `modern` template.

## Un exemple de diagramme de flux

```mermaid
flowchart LR

A[Hard] -->|Text| B(Round)
B --> C{Decision}
C -->|One| D[Result 1]
C -->|Two| E[Result 2]
```
